Hold Shift and Check Checkboxes

  • Project Synopsis

    In this lesson I select all of the checkboxes on the page (document.querySelectorAll('.inbox input[type="checkbox"]')). I also initialize a 'lastchecked' variable.

    For each checkbox I listen for a 'click' and fire handleCheck(e).

    When running through handleCheck(e) I initialize an 'inBetween' variable as false. I then check if the user is holding the shift key and that they are checking the input (e.shiftKey && this.checked). If that is the case, then for each checkbox I check if the checkbox is either the current input, or the 'lastChecked' item. If that is the case, I flip the 'inBetween' polatity (inBetween = !inBetween). I also check if 'inBetween' is true, and if so, I check that checkbox.

    Finally I set 'lastChecked' to the current checkbox 'this'.
