Fun with HTML5 Canvas

  • Project Synopsis

    In this lesson I am drawing a canvas onto the page using a 2d context. The size of the canvas is determined by the size of the parent container.

    Inside of the context of the canvas I assign initial values for: 'strokeStyle', 'lineJoin', 'lineCap', 'lineWidth' and 'globalCompositeOperation'. These help define the look of the brush stroke. I then initialize variables that will determine if the user is drawing, their last X and Y positions, the current hue, and the direction that determines if we're increasing or decreasing the brush stroke size.

    I'm listening on the <canvas> element for 'mousedown', mousemove', 'mouseup' and 'mouseout' events.

    'mousedown' - Pass the event into a function (e), set 'isDrawing' to true and set the 'lastX' and 'lastY' values to match the current values from the event 'e.offsetX' and 'e.offsetY'.

    'mousemove' - Pass the event into a function Draw(e). If 'IsDrawing' is not true, exit the function. Otherwise, set the context 'strokeStyle' to inject the 'hue' variable using template strings. I then begin a path in the context (.beginPath()). start the path at 'lastX' and 'lastY' (.moveTo()). Draw a line to the current 'e.offsetX' and 'e.offsetY' (.lineTo()). Then I fill out the line stroke (.stroke()).

    Next I increment the 'hue' value, if it's >= 360, I reset it to 0 to loop back around on the colour wheel. I also increase or decrease the 'lineWidth' in a range between two set values, If the 'lineWidth' hits the bounds, I flip the polarity of 'direction' and increment 'lineWidth' in the opppsite direction.

    'mouseup' and 'mouseout' - Set 'isDrawing' to false.
