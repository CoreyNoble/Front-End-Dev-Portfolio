Array Cardio Day 2

  • Project Synopsis

    In this lesson I learned about even more methods that can alter an array.

    .some() - Method to check if one of the items is true. In this example we checked if one person is >= 19.

    .every() - Method to check if all of the items are true. In this example we checked if every person is >= 19.

    .find() - Find is like filter, but instead of returning a subset of the array, it is going to return the first one it finds. In this example we find a comment with a specific ID (comment.id === 823423).

    .findIndex() - Finds the index value of the item in an array. In this example we find the index of the comment ID that we want to remove.

    ... spread - Spreading items into an array. In this example we spread all of the comments back into the array and slice out the 'index' that we want to remove.
