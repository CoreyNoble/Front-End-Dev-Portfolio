Flex Panel Gallery

  • Project Synopsis

    In this lesson I create an interactive motivational gallery.

    First I find all of the elements with a class of '.panel', and on each item I listen for a 'click' event. When the item is clicked I togge a class '.open' to manipulate the size of that '.panel'.

    I also listen for a 'transitionend' event on each '.panel' because when the animation for the panel opening is finished, I want to toggle a second class that animates the text. When the function runs, I pass in the event (e) and check if the animation that finished was the 'flex' animation 'e.propertyName.includes('flex')'. If so, I toggle the second class '.open-active'.
