Sticky Nav

  • Project Synopsis

    In this lesson I create a sticky navigation that starts partway down the page but sticks to the top when the <div> hits the top of the page.

    First I grab the <div> 'nav' and get the top of the <div> (topOfNav = nav.offsetTop). Then I listen on the 'window' for a 'scroll' event and fire fixNav().

    In fixNav() I look for if (window.scrollY >= topOfNav), If true, I add a class of '.fixed-nav' to the element to give it fixed positioning on the top of the page. I also offset the page by giving it padding on top, equivelant to the offset height of the <div> (page.style.paddingTop = nav.offsetHeight + 'px'. If false, I remove the '.fixed-nav' class and remove the padding that was added to the top of the page.
