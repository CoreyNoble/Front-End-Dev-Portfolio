Key Sequence Detection

  • Project Synopsis and Code

    In this lesson I learned how to listen for a specific combination of inputs before firing a function.

    First I initialize an empty array 'pressed', as well as a 'secretCode' variable.

    I listen for a 'keyup' event on the 'window' object. I pass the event into a function ((e) => {}).

    Inside of the function I push the 'key' event into the 'pressed' array. I then trim the array to ensure it is never longer than the maximum number of characters that 'secretCode' contains.

    Next I join the 'pressed' array together into one string (.join('')) and check if it includes 'secretCode' (.includes(secretCode)), if so, the 'secretCode' is detected and the function can proceed.
