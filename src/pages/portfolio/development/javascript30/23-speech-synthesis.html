Speech Synthesis

  • Project Synopsis

    In this lesson I learn how to convert text to a new speech synthesis utterance from the browser.

    First I create a new speech synthesis utterance (const msg = new SpeechSynthesisUtterance()). I get the following elements: <select>, All <input>, 'speak' and 'stop' <button>. I initialize an empty 'voices[]' array, as well as set what 'msg' text will be spoken by grabbing the [name='text'] element.

    I listen for the following events:

    • speechSynthesis

      • 'voiceschanged', populateVoices()
      • This event is needed before populating the voices in-order to give the browser enough time to have the required event ready.

    • <select>

      • 'change', setVoice()

    • <option>

      • 'change', setOption()

    • speakButton

      • toggle
      • This will start the speech synthesis.

    • stopButton

      • 'click', () => toggle(false)
      • This will stop the speech synthesis.

    In populateVoices() - First to get the 'voices' we .getVoices() from 'this' (speechSynthesis). We set the .innerHTML of <select> to the list of 'voices'. Then we .filter() over each voice to only include voices with a language of 'en'. Next we .map() over each voice to add <option> HTML to each entry. Lastly we .join('') to convert the array into one string.

    In setVoice() - First we set 'msg.voice' to the 'voice' that macthes the <select> item (msg.voice = voices.fint(voice => voice.name === this.value)). Then we fire toggle().

    in toggle(true/false) - First we stop any currently playing speech (speechSynthesis.cancel();). Next we check if either true or false was passed into the function. If false, do nothing more. If true (default), restart the speech synthesis (speechSynthesis.speak(msg);).

    In setOption() - This sets the 'msg' voice to the <option> that was selected. Then we fire toggle().
