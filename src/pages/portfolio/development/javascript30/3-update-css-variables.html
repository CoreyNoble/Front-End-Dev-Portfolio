Update CSS Variables

  • Project Synopsis

    In this lesson I learned how to update an elements style attribute by injecting variable values using es6 template strings (`${this.name}`).

    First I get all of the inputs on the page with a class of '.controls' (document.querySelectorAll('.controls input')).

    I listen for either a 'change' or 'mousemove' event on each input (inputs.forEach(input => input.addEventListener('change', handleUpdate))).

    When the event triggers, the function that fires (handleUpdate()) modifies the style that matches the element name that triggered the event (.style.setProperty(`--${this.name}`, this.value + suffix).

    The suffix checks for a data attribute on the element (this.dataset.sizing) and appends it to the style modification.
