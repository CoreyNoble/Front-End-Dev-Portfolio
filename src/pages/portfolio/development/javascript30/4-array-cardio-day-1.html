Array Cardio Day 1

  • Project Synopsis

    In this lesson I learned about various methods that can alter an array.

    .filter() - I filter the list of inventors to get back those whore were born in the 1500's.

    .map() - .Takes in an array, does something with that array and returns a new array of the same length. Like a factory machine that takes in a material, modifies it and outputs the modified material. In this example I .map() an array of full names from the list of inventors.

    .sort() - Takes in an array, compares two items, and pushes the item either up or down, depending on the comparison. In this example I sort the inventors by birthdate, from oldest to youngest.

    .reduce() - Takes in an array and returns only what is specified. In this example I reduce the inventors array to return how many years all of the inventors lived for.

    The lesson then followed up with an excercise for each method.

    Using .map() and .filter() I take in a list of <a> elements from a wikipedia page. I create a .map() for all of the <a> 'textContent' and .filter() out any items that .includes('de').

    Using .sort() I take in an array of 'people' and .sort() them alphabetically after splitting the string .split(', ') into First and Last name variables.

    Using .reduce() I create a variable object that takes in a data object[] of vehicle types, and an item (obj, item). I initialize the constant as an empty object, then for each item in the data, if it does not exist, I initialize that item and its counter into the variable object. I then increment the count and return the object. This means I can count the amount of times a unique item is used in a data object. It is also future proofed because the item is initialized as it is passed into the variable, so it doesn't matter what type of data will be coming through the data object, all unique instances will be tracked.
