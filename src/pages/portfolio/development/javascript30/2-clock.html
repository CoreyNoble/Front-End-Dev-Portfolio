Clock

  • Project Synopsis

    In this lesson I run a function every second (setInterval(setDate, 1000)).

    Every time the function runs, I store the current time (const now = new Date()). Using the current time we get the time in seconds (now.getSeconds()), minutes (now.getMinutes()) and hours (now.getHours()). We then calculate the degree of rotation for each hand, offset by +90 to ensure the rotation of the element starts north.

    const secondsDegrees = ((seconds / 60) * 360) + 90;
    const minutesDegrees = ((minutes / 60) * 360) + 90;
    const hoursDegrees = ((hours / 12) * 360) + 90;

    I query the document for each hand, and rotate each hand to the matching degree value using es6 template strings to inject the value into the elements 'style' attribute (secondHand.style.transform = `rotate(${secondsDegrees}deg)`).
