Drum Kit

  • Project Synopsis

    In this lesson I am passing the event information (e) from 'keydown' on the 'window' object into a playSound(e) function.

    I then query the 'document' for two types of elements. The <audio> element, and elements that have a class of '.key'. I checked if the event 'keyCode' (e.keyCode) matches the data attribute 'data-key' of the element. If the <audio> element does not exist I exit the function.

    Next I reset the audio time to 0 (audio.currentTime = 0). I then play the <audio> (audio.play();). I also add a class of '.playing' to the '.key' element to animate the UI.

    I also am listening for when the animation is finished playing on the element 'transitionend', and then fire a function to remove the class of '.playing' from the '.key'.
