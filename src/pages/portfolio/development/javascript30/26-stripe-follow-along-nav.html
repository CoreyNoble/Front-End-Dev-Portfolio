Stripe Follow Along Nav

  • Project Synopsis

    In this lesson I improve upon the puppy <div> functionality from the 'Follow Along Link Highlighter' lesson by creating a flexible dropdown navigation in-which the size of any dropdown item can be dynamic. As the user hovers on each dropdown item, the puppy <div> follows along in both size and position to transition into the background for that newly shown dropdown item.

    First I get the 'nav' and the 'triggers' (<li>) as well as the puppy <div> 'background'.

    Then I listen for:

    • triggers.forEach(trigger => trigger.addEventListener)
      • 'mouseenter', handleEnter()
      • 'mouseleave', handleLeave()

    I use an arrow function to keep <li> as 'this'.

    In handleEnter() - First add a class of '.trigger-enter' to the trigger. This class changes 'display:none' to 'display:block'. The class is added on a delay, so I use an arrow function to check if the class is there. If so, I add a second class of '.trigger-enter-active'. This is known as stepping in an animation. First display it, then transition it.
    (setTimeout(() => this.classList.contains('trigger-enter' && this.classList.add('trigger-enter-active', 150);).

    Next I add a class of '.open' to 'background'. I get the 'dropdown' and its' bounding rectangle, I also get the bounding rectangle of the 'nav'. Using the 'dropdown' bounding rectangle, I calculate the 'width' and 'height values that 'background' will need. Using 'nav' and 'dropdown' bounding rectangles I calculate the position that 'background' will need. I also use 'nav' as an anchor for positioning and adjusting the 'top' and 'left' accordingly using the 'dropdown' coordinates.

    const coords = {
      width: dropdownCoords.width,
      height: dropdownCoords.height,
      top: dropdownCoords.top - navCoords.top,
      left: dropdownCoords.left - navCoords.left
    }

    Finally I set the 'background' styles by injecting the 'coords' into the style attributes of 'background' using es6 template strings. (background.style.setProperty('width', `${coords.width}px`)).

    In handleLeave() - Hide the elements. Remove the classes '.trigger-enter' and '.trigger-enter-active' from the trigger. Remove the class of '.open' from 'background'.
