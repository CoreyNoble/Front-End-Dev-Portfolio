Whack A Mole

  • Project Synopsis and Code

    In this lesson I create a whack-a-mole game.

    First I get the following elements: 'holes', 'scoreBoard', 'moles' and 'startCTA'. I also initialize 'lastHole', 'timeUp = false' and 'score = 0'.

    I listen for the following events:

    • moles.forEach(mole => ())
      • 'click', bonk(e)
    • 'startCTA'
      • 'click', startGame()

    Functions

    In startGame() - First I Initialize the game values: 'scoreBoard.textContent = 0', 'timeUp = false', 'score = 0'. Then I fire peep(). I also set the duration of the game (setTimeout(() => timeUp = true, 10000)).

    In peep() - First I set a random 'time' that each mole will pop up for. Then I choose a random 'hole' inside of 'holes'. I add a class of '.up' to that hole which pops up the mole. Check if 'time' is up, if not, run peep() again. After 'time' is up (setTimeout(() {}, time) remove the class of '.up'.

    In randomTime(min, max) - A function that takes in a 'min' and 'max' paramater, and returns a random value between the 'min' and 'max' bounds.

    In randomHole(holes) - Gets a random DOM element using a node list 'holes'.

    In bonk(e) - First I check if the event is trusted, if not I exit the function (!e.isTrusted). This makes sure the clicks are genuine and are not generated by an external source. If the click is genuine, increase the 'score', remove the class of '.up' from the element, and update the 'score' in the 'scoreboard'.
